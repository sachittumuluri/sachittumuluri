## ⚡ about

I build software where **signals meet decisions** — machine-learning systems, quantitative research tooling, and the occasional beautifully over-engineered side project.

- 🛠️ Currently building **Vega Labs** — AI agents for sales pipelines that keep judgment in the model and rules in code
- 📈 Quant research with **MAIF** — backtesting infrastructure, strategy validation, reproducible scorecards
- 🫀 Applied **topological data analysis** to ECG signals to detect ventricular tachycardia
- 🎧 Shipped a noir hardware-deck music player for macOS, because tools should have a soul
- 🎓 UMass Amherst · previously @jpmorganchase & @PimBrands
